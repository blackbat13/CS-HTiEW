Python - Problem n królowych

Funkcja check_new_position sprawdza, czy możemy ustawić nową królową na podanej pozycji (x,y).
Instrukcja if positions[i] == y: sprawdza, czy poprzednia królowa jest już ustawiona w wierszu y.
Instrukcja if y - positions[i] == x - i: sprawdza, czy poprzednia królowa atakuje sprawdzaną pozycję diagonalnie.
