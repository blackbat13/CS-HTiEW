Python - Wprowadzenie

Wprowadzenie do języka Python 3.

Uwaga

To nie jest tutorial programowania. Założenie jest takie, że wiesz już co nieco o programowaniu, być może znasz inny język programowania i chcesz się dowiedzieć jak podstawowe konstrukcje wyglądają w języku Python 3.

Nie będziemy tutaj tłumaczyć czym jest pętla czy instrukcja warunkowe, a jedynie pokażemy, jak te konstrukcje wyglądają w języku Python 3.

Przygotowanie

Instalacja interpretera

Pobieramy interpreter języka Python 3 w jak najnowszej wersji z następującej strony (przycisk "Download Python" na samej górze strony).

Instalacja środowiska

Pobieramy darmowe środowisko Pycharm Community Edition. Pamiętamy, aby wybrać wersję przeznaczoną dla naszego systemu operacyjnego. Instalujemy postępując zgodnie z instrukcjami instalatora.
